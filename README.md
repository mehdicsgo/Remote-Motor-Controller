# ریموت کنترلر موتور dc بر پایه stm32
# کنترل حلقه باز ( بدون فیدبک )
ماژول ها و قطعات پروژه :
۱. برد stm32f07ve
۲. ماژول درایور موتور BTS7960
۳ . ماژول بولوتوث HC_05
۴. موتور ۵ ولت dc
۵. نرم افزار android serial terminal (و نرم افزار های مشابه برای ارسال دستورات)

# روش کار
منطق برنامه به این صورت است که کاربر با ارسال دستورات ۵ بایتی به فرمت x,yzk
که x یک عدد یک رقمی در مجموعه { ۰ ، ۱ ، ۲ } میباشد که نشان دهنده وضعیت موتور
به شکل زیر میباشد :
۱ = حرکت ساعت گرد 
۲ = حرکت پاد ساعتگرد 
۰ = موتور خاموش
میباشد همچنین ارقام yzk یک عدد سه رقمی در بازه ۲۰۰_۹۹۹ بوده که مقدار pwm و درنتیجه 
دیوتی سایکل موتور را مشخص می‌کند که توسط اپلیکیشن واسط و از طریق ارتباط بولوتوث
به میکرو ارسال می‌شود.
